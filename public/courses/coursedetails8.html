Course overview

This learning path is a complete guide to building scalable web applications using Facebook’s popular ReactJS library, leveraging the power of a virtual DOM to create fast, fluid UIs. An overview of the library is given and then a small application is developed using the ‘Thinking in React’ steps recommended by Facebook. Learners will be able to create components to create static versions of applications. Using Props, State (using Hooks) and inverse data flow, a fully functioning application will be created. Techniques to use external data sources is explored, along with relevant lifecycle hooks. Creating a single page application through the use of routing is covered. Finally, state management tools such as Context and Reducers are introduced as a way of handling application data in large scale applications.
